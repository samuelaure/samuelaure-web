# Samuel Auré Web
Multi-dimensional personal brand platform for Samuel Auré. Founder, Partner, Creator, and Operator.

## 🚀 Overview
A high-fidelity web platform that balances personal philosophy with technical systems. It serves as a central hub for Samuel Auré's professional ecosystem, bridging life coaching, venture building, and tech partnerships.

## 🛠 Tech Stack
- **Framework**: React 18+ (Vite)
- **Language**: TypeScript
- **Styling**: Vanilla CSS (CSS Variables for Dynamic Themes)
- **Icons**: Lucide React
- **Routing**: React Router DOM (v6+)
- **Integration**: Google Calendar Appointment Schedule

## 🏗 Architecture
The site is built as a highly optimized Single Page Application (SPA) with a focus on narrative flow and conversion.

### Core Modules
- **Theme Engine**: Handles the switch between "Day" (Philosophy) and "Night" (Tech) modes.
- **Dimension Grid**: A clean, modular presentation of the 4 core business pillars.
- **Plan Maestro Integration**: Narrative-led conversion funnel for the private inner circle.
- **Booking System**: Unified scheduling via `/hablemos` and global modal.

## 🏃‍♂️ Quick Start
```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
