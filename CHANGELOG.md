## [0.1.0] - 2026-03-09

### Added
- **Freedom Engineering Brand**: Established the "Ingeniería de Libertad" narrative for Samuel Aure.
- **Dual-Theme Engine**: Implemented a "Life" (Day) and "Tech" (Night) theme system using CSS Variables.
- **Booking Infrastructure**: Integrated Google Calendar Appointment Schedule via a global modal and a dedicated `/hablemos` route.
- **Plan Maestro Funnel**: Created a high-conversion landing page structure focused on the Plan Maestro private group.
- **Partner Stack**: Defined the 4 core dimensions (Mindset, Tech, Growth, Capital) as a unified partnership proposal.
- **Premium UI/UX**: Added reveal animations, responsive typography (clamp), and refined spacing for an elite personal brand feel.
- **Scaffolding**: Complete Vite + React + TypeScript + Vanilla CSS architecture setup.
- **Identity Assets**: Added a custom minimal favicon.
